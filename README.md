# Robot Arm Kinematics Simulation 🦾

A Python simulation of a 2D robotic arm using 
Forward and Inverse Kinematics with real-time animation.

## What it does
- Simulates a 2-link robotic arm reaching target positions
- Implements Forward Kinematics to calculate end position
- Implements Inverse Kinematics to calculate joint angles
- Animates arm movement with path tracing in real time

## Tools Used
- Python 3.12
- NumPy
- Matplotlib

## How to Run
pip install numpy matplotlib
python robot_arm.py
